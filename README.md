# I'm Booked
The backend of an appointment booking system for businesses.

## Tech Stack
- Backend: Spring Boot 3
- Database: MySQL
- Migrations: Flyway

## Setup
1. Clone the repo
2. Copy `env.properties.example` to `env.properties` and fill in values
3. Do the same for `flyway.conf.example` and `flyway.conf`
4. Create MySQL database: `CREATE DATABASE im_booked`
5. Run the application
